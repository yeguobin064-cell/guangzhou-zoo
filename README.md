# 广州动物园智能路线规划系统

## 项目简介

本系统是一个专为广州动物园定制的景区路线规划系统，采用C语言实现，基于图论算法（Dijkstra、DFS、Floyd等），支持路径查询、路线规划、景点CRUD管理等功能。

## 功能列表

### 查询功能
1. 查看园区地图 - 显示所有景点分布及连接关系
2. 查看景点列表 - 显示所有景点信息
3. 查看景点详情 - 查看指定景点的详细信息
4. 查询最短路径 - 两点间最短路径（Dijkstra算法）
5. 规划游览路线 - 时间限制下的最佳路线（贪心算法）
6. DFS导游线路 - 深度优先遍历生成导游线路
7. 检测回路 - 检测导游线路中的回路
8. 查看所有路径 - 所有景点间最短路径

### 管理功能
9. 添加景点 - 支持录入新景点信息
10. 删除景点 - 移除不需要的景点
11. 查询景点 - 按名称或类型搜索
12. 修改景点 - 更新景点详细资料

### 智能推荐
13. 随机路线 - 随机生成含公共厕所的游览路线

## 技术栈

- **语言**: C11
- **构建**: CMake 3.16+ / Ninja
- **算法**: Dijkstra、Floyd、DFS、Fisher-Yates洗牌
- **数据结构**: 邻接表

## 构建与运行

```bash
# 创建构建目录
mkdir build && cd build

# 生成构建文件
cmake .. -G Ninja

# 编译
ninja

# 运行
.\bin\zoo_route.exe
```
