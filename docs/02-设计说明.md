# 02 - 设计说明

## 1. 模块架构

```
main.c          主程序入口、菜单循环、用户交互
graph.c         图的基本操作（创建、遍历、增删改查、展示）
pathfinding.c   路径规划算法（Dijkstra、Floyd、DFS、贪心、随机路线）
zoo_data.c      广州动物园初始化数据
consoleio.c     终端界面（颜色、输入输出、控制台设置）
```

## 2. 数据结构设计

### 2.1 邻接表图结构（ALGraph）

```c
typedef struct {
    Vnode roadList[MAX_VERTEX_NUM];  // 顶点数组
    int numVertex;                   // 顶点数量
    int numEdge;                     // 边数量
} ALGraph;
```

### 2.2 顶点节点（Vnode）

```c
typedef struct Vnode {
    char name[MAX_NAME_LEN];  // 景点名称
    SpotInfo info;            // 景点详细信息
    CList first;              // 邻接链表头指针
} Vnode;
```

### 2.3 邻接边节点（CNode）

```c
typedef struct CNode {
    int index;           // 邻接点下标
    int length;          // 距离（米）
    int time;            // 步行时间（分钟）
    struct CNode* next;  // 下一条边
} CNode;
```

### 2.4 景点信息（SpotInfo）

```c
typedef struct {
    char name[MAX_NAME_LEN];
    char description[MAX_DESC_LEN];
    SpotType type;        // 入口/展馆/景点/设施/科普
    int visitTime;        // 游览时间（分钟）
    int popularity;       // 受欢迎程度 1-10
    Animal animals[MAX_ANIMAL_NUM];
    int animalCount;
} SpotInfo;
```

### 2.5 路径结果（PathResult）

```c
typedef struct {
    int path[MAX_VERTEX_NUM];  // 路径顶点数组
    int pathLength;            // 路径长度
    int totalDistance;          // 总距离（米）
    int totalTime;             // 总时间（分钟）
    int spotCount;             // 经过景点数
} PathResult;
```

## 3. 景点类型枚举

```c
typedef enum {
    TYPE_ENTRANCE = 0,    // 入口
    TYPE_ANIMAL_HOUSE,    // 动物展馆
    TYPE_SCENIC_SPOT,     // 景点
    TYPE_FACILITY,        // 设施
    TYPE_EDUCATION        // 科普教育
} SpotType;
```

## 4. 模块间依赖关系

```
main.c
  ├── global.h      （全局常量、数据结构定义）
  ├── graph.h       （图操作接口声明）
  └── consoleio.h   （终端界面接口声明）

graph.c       → global.h, graph.h, consoleio.h
pathfinding.c → global.h, graph.h, consoleio.h
zoo_data.c    → global.h, graph.h
consoleio.c   → global.h, consoleio.h
```

## 5. 文件数据格式（zoo_layout.txt）

```
<景点数> <边数>
景点1名称
景点2名称
...
起点 终点 距离
起点 终点 距离
...
```

## 6. 错误处理策略

- 空指针检查：所有公共函数入口均检查 NULL
- 数组越界：顶点数量不超过 MAX_VERTEX_NUM（50）
- 未找到景点：LocateVertex 返回 -1
- 不可达路径：Dijkstra 返回 NOT_FOUND
- 操作确认：删除景点前要求用户确认
